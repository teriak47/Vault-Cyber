# Types de Supports Réseau

La communication est transmise sur un [[Network|réseau]] par le biais de [[NetworkMedia|supports]]. Celui-ci fournit le [[CommunicationChannel|canal]] via lequel le message se déplace de la source à la destination.

Les réseaux modernes utilisent principalement trois types de médias pour interconnecter les appareils:

*   **[[CopperWire|Fils métalliques dans les câbles]]** - Les données sont encodées en [[ElectricalPulses|impulsions électriques]].
*   **[[FiberOpticCable|Fibres de verre ou de plastique]] à l'intérieur des câbles ([[FiberOpticCable|câble à fibres optiques]])** - Les données sont codées en [[LightPulses|impulsions lumineuses]].
*   **[[WirelessTransmission|Transmission sans fil]]** - Les données sont codées via la modulation de fréquences spécifiques d'[[ElectromagneticWaves|ondes électromagnétiques]].

## Critères de Choix d'un Support Réseau

Les quatre principaux critères de choix d'un support réseau sont:

*   Quelle est la distance maximale sur laquelle les supports peuvent transporter correctement un signal?
*   Quel est l'environnement dans lequel les supports seront installés?
*   Quelle est la quantité de données et à quelle vitesse doit-elle être transmise?
*   Quel est le coût des supports et de l'installation?

## Câbles Réseau Courants

Les trois câbles réseau les plus courants sont les câbles à [[TwistedPairCable|paire torsadée]], les [[CoaxialCable|câbles coaxiaux]] et les câbles à [[FiberOpticCable|fibres optiques]].

*   **[[TwistedPairCable|Câbles à paire torsadée]]**: La technologie [[Ethernet|Ethernet]] utilise généralement ce type de câble pour interconnecter les appareils.
*   **[[CoaxialCable|Câbles coaxiaux]]**: Il s'agit du type de câble cuivré servant aux sociétés de télévision câblée et pour connecter les composants des systèmes de communication par satellite.
*   **[[FiberOpticCable|Câble à fibres optiques]]**: Ce câble (en verre ou en plastique) peut transporter des informations numériques à très grande vitesse sur de longues distances. Comme il utilise la lumière à la place de l'électricité, les [[ElectromagneticInterference|interférences électriques]] n'affectent pas le signal.
