# Maîtriser la Recherche Sur Internet

Chaque jour, des millions de recherches sont effectuées, mais parfois ce dont vous avez besoin, c'est comme chercher une aiguille dans une botte de foin. C'est là qu'interviennent les fonctions de recherche. Voyons comment cela fonctionne et comment vous pouvez trouver tout ce dont vous avez besoin sur Internet.

# Accéder à Un [[SearchEngine|Moteur De Recherche]]

Tout d'abord, vous devez accéder à un [[SearchEngine|moteur de recherche]]. Il s'agit de sites web qui vous permettent de rechercher les éléments que vous souhaitez et vous obtiendrez une page complète de résultats potentiellement utiles en fonction du texte saisi.

# Configuration Du [[SearchEngine|Moteur De Recherche]]

Lorsque vous ouvrez le navigateur, il est possible que le [[SearchEngine|moteur de recherche]] par défaut soit déjà affiché. Si ce n'est pas le cas, saisissez l'adresse du [[SearchEngine|moteur de recherche]] dans la barre d'adresse.

Vous pouvez également modifier le [[SearchEngine|moteur de recherche]] par défaut dans le menu des paramètres. Cliquez ou appuyez sur Paramètres, puis sur [[SearchEngine|Moteur de recherche]], puis sélectionnez votre option préférée.

# Effectuer Une Recherche De Base

Maintenant, saisissez quelque chose dans la barre de recherche et appuyez sur Entrée ou appuyez sur OK. Vous obtenez une liste des sites web considérés comme pertinents par le [[SearchEngine|moteur de recherche]] exemple: **best hard hat 2024**, ainsi que les recherches associées.

Vous pouvez également spécifier vos recherches par type. Vous pouvez rechercher des images, des vidéos ou des résultats d'achat. Faisons des tests plus avancés.

# Recherche Avec Guillemets

## Utilisons Les Guillemets

Lorsque vous recherchez une expression exacte ou un terme spécifique, placer votre requête entre guillemets indique au [[SearchEngine|moteur de recherche]] d'extraire les pages contenant exactement ces mots.

exemple: "best red hard hat"

# Exclure Des Mots De la Recherche

## Ensuite, En Excluant Les Mots

En plaçant un signe moins devant un mot, vous l'excluez des résultats de recherche. C'est parfait pour réduire les résultats lorsque vous êtes bombardé avec des informations non pertinentes.

exemple: best -red hard hat 2024

# Recherche Sur Un Site Spécifique

## Concentrer Sur Un Seul Site Web

En concentrant votre recherche sur un seul site web, vous obtenez souvent de meilleurs résultats.

**exemple:site:**

Cette fonction est d'une valeur inestimable pour la recherche universitaire ou lorsque vous faites confiance à une source d'informations fiables.

exemple:site:dursafetybest red hard hat 2024

# Recherche Par Type De Fichier

## Vous Recherchez Des Types De Fichiers Spécifiques ?

**filetype:** est votre opérateur préféré. Qu'il s'agisse de PDF, de fichiers PPT ou de DOC, cette fonction rationalise la recherche de ressources téléchargeables.

exemple: filetype:pdf best hard hat 2024

# Filtrer Par Période

Vous pouvez également spécifier jusqu'où vous voulez remonter la recherche. Accédez à Outils ou Outils de recherche et sélectionnez une période.

Si vous maîtrisez ces opérateurs de recherche, vous trouverez exactement ce que vous cherchez rapidement et précisément.
