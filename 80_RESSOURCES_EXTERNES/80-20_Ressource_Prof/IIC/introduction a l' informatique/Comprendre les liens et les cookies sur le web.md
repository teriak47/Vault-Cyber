# Comprendre Les Liens Et Les [[HttpCookies|cookie]] Sur Le Web

Une introduction essentielle à la navigation web et à la gestion des données personnelles en ligne.

## Les Liens : Votre Porte D'entrée Vers Le Web

Lorsque vous visitez un site web, vous y voyez beaucoup de choses qui ressemblent à ceci.

Parfois, il s'agit de texte en bleu avec un souligné, mais en général, c'est du texte qui a un style différent des autres.

Lorsque vous placez le pointeur dessus, le pointeur passe d'une flèche à une main.

**Il s'agit de liens, et ils sont cliquables.**

En d'autres termes, si vous cliquez dessus, il se passera quelque chose. La page web peut ouvrir une liste déroulante, ou charger une image ou une vidéo, mais la plupart du temps, vous quittez cette page et chargez une autre page ou un autre site web.

## Navigation Entre Les Pages

-   Cliquez sur ce lien.
-   Il charge une autre page du même site web.
-   Pour revenir à la page que vous venez de quitter, cliquez sur le bouton Précédent.
-   **Au revoir.** Si vous cliquez sur le bouton Précédent, actualisez la page précédente, puis souhaitez revenir à la nouvelle page, pouvez cliquer à nouveau sur le même lien, ou vous pouvez cliquer sur le bouton Suivant.

## Première Rencontre Avec Les [[HttpCookies|cookie]]

Si vous quittez un site web pour aller sur un autre site et que c'est la première fois que vous visitez ce site web, il vous sera demandé de sélectionner vos préférences en matière de cookies.

Les cookies sont de petits fichiers de données que les sites web envoient à votre terminal, sur votre ordinateur, votre smartphone ou votre tablette, chaque fois que vous les visitez.

Considérez-les comme de petites notes que le site web laisse sur votre terminal afin qu'il se souvienne de vous et de vos préférences lors de votre prochaine visite.

## Pourquoi Les Cookies Sont-ils Importants ?

**Les cookies sont importants, alors prenons une minute pour les comprendre.**

Ils sont utilisés pour plusieurs choses différentes.

1.  **Mémorisation des préférences**
    Sauvegarde de vos choix de langue et thème
2.  **Analyse du comportement**
    Compréhension de votre utilisation du site
3.  **Commerce en ligne**
    Conservation du contenu de votre panier
4.  **Publicité ciblée**
    Suivi pour des annonces personnalisées

### Mémorisation De Vos Préférences

Lorsque vous visitez un site web, il peut vous demander de vous connecter, de choisir une langue ou sélectionner un thème.

Un cookie enregistre ces choix. Ainsi, la prochaine fois que vous visitez le site, il mémorise vos préférences sans que vous ayez à les saisir à nouveau.

C'est comme dans un café, où le personnel connaît votre commande, car vous venez régulièrement.

### Personnalisation De Votre Expérience

Les cookies permettent aux sites web de comprendre comment vous les utilisez.

Par exemple, si vous visitez fréquemment certaines pages d'un site, les cookies peuvent aider le site web à suggérer d'autres contenus susceptibles de vous intéresser.

Si vous allez dans une librairie et que vous passez quelques minutes à parler au propriétaire, il sera probablement en mesure de faire de très bonnes suggestions pour les livres qui vous intéressent.

### Commerce En Ligne Et Publicité

#### Boutiques En Ligne

Pour les boutiques en ligne, les cookies sont essentiels. Ils se souviennent de ce que vous avez ajouté à votre panier, même si vous quittez la page.

**Sans les cookies, votre panier serait réinitialisé à chaque fois que vous cliquiez sur un nouveau lien.**

#### Publicité Ciblée

Les annonceurs utilisent des cookies pour suivre les sites web que vous visitez afin qu'ils puissent vous montrer les annonces susceptibles de vous intéresser.

C'est comme voir des panneaux d'affichage pour produits que vous avez récemment parlé ou examiné.

## Préoccupations En Matière De [[Confidentiality|Confidentialité]]

Bien que la plupart des utilisateurs soient satisfaits que les cookies mémorisent leurs préférences, beaucoup de gens n'aiment pas l'idée que les annonceurs les suivent sur Internet pendant la navigation.

**Imaginez qu'une agence de publicité vous suive au centre commercial, en notant chaque article que vous regardez, combien de temps vous le regardez, et si vous l'achetiez ou non.**

Bien que les cookies soient très utiles, ils ont soulevé des préoccupations en matière de [[Confidentiality|confidentialité]], car ils peuvent suivre une grande partie de ce que vous faites en ligne.

## Prendre Le Contrôle De Vos Cookies

Les réglementations et les mises à jour des navigateurs vous permettent de mieux contrôler les cookies, vous pouvez les bloquer ou les supprimer si vous le souhaitez.

**C'est pourquoi nous devons effectuer ces sélections la première fois que nous visitons un site web.**

En fin de compte, chaque site web est différent en matière de cookies. Certains sont plus directs que d'autres, alors cela vaut la peine de passer un moment de lecture sur ce que le site web prévoit de faire de vos informations.

Si vous n'aimez pas ce que vous lisez, vous n'êtes pas obligé de l'accepter. Vous pouvez refuser les cookies du site web.

La plupart des sites web vous autorisent à consulter leurs pages sans accepter les cookies, mais d'autres pas. Si c'est le cas, vous devez décider dans quelle mesure vous souhaitez consulter ce site web. Il est peut-être préférable de trouver les informations ailleurs.
